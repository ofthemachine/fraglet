# Release v0.1.0

## Key Highlights
Initial release of fraglet-entrypoint - a container entrypoint for injecting and executing fraglet code.

- **Fraglet Injection**: Inject fraglet code into target files using single match or range-based substitution
- **Configurable Entrypoint**: Support for custom configuration via `fraglet-entrypoint.yaml` or `FRAGLET_CONFIG` environment variable
- **Documentation Commands**: Built-in `agent-help` and `how-to` commands for container documentation
- **Multi-arch Support**: Binaries available for linux/amd64 and linux/arm64

## Quick Start

```bash
# Download the binary for your platform
# Extract and place in your container

# Create a fraglet-entrypoint.yaml configuration
cat > fraglet-entrypoint.yaml <<EOF
fragletTempPath: /FRAGLET
injection:
  codePath: /code/script.sh
  match: FRAGLET
execution:
  path: /code/script.sh
  makeExecutable: true
EOF

# Use as container entrypoint
ENTRYPOINT ["/usr/local/bin/fraglet-entrypoint"]
```
