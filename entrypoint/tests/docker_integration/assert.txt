# Container Usage

## Mount and Run

Mount your fraglet code to /FRAGLET and run:

```bash
docker run --rm -v /path/to/fraglet:/FRAGLET:ro <container>
```

## Code Injection

Code will be injected into `/code/hello-world.sh` at the marker: `FRAGLET`

## Execution

After injection, the container executes: `/code/hello-world.sh`

## Example

Here's a functional example using the existing code:

```bash
# Read the existing code
cat > /tmp/fraglet.sh << 'EOF'
echo "FRAGLET replacing here"
EOF

# Mount and run
docker run --rm -v /tmp/fraglet.sh:/FRAGLET:ro <container>
```

# Read the existing code

#!/bin/bash

# Mount and run
docker run --rm -v /tmp/fraglet.sh:/FRAGLET:ro <container>
